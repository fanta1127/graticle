# Graticle - University Goukon Service

大学生限定の合コンマッチングサービス「Graticle」のLPおよびHPです。

## 概要

- **LP (`index.html`)**: サービスの紹介、特徴、料金、参加申し込みへの導線。
- **About (`about.html`)**: 運営者情報（神戸大学起業部 graticle）。

## 特徴

- **大学生限定**: 安心・安全な出会いを提供。
- **友達と参加**: 1対1ではなく、グループでの参加を推奨。
- **プレミアムデザイン**: 信頼感と若々しさを兼ね備えたデザイン。

## 技術スタック

- HTML5
- CSS3 (Variables, Flexbox, Grid, Glassmorphism)
- JavaScript (Smooth Scroll, Intersection Observer)
